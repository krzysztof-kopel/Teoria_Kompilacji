# Projekt — interpreter prostego języka programowania

Interpreter prostego języka programowania ze wsparciem dla macierzy, tworzony w ramach przedmiotu Teoria Kompilacji w semestrze zimowym roku akademickiego 2025/26. Projekt bazowany na bibliotece SLY.

W ramach projektu powstały:
- skaner i tokenizer
- parser
- generator abstrakcyjnego drzewa składni
- type checker
- interpreter


## Jak używać?
Należy uruchomić plik main.py z folderu src i podać nazwę programu z tests, który ma zostać uruchomiony.
